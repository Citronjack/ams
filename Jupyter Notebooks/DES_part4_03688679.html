4.2.1: Auto Correlation¶

First, we need to understand what correlation acutally means: It measures the linear dependency between random variables, or in case of the autocorrelation between itself and a later point in time. The correlation is bounded in the interval [-1,1] where the edges mean negative or positive linear relationship respectively. Now to my experiment.

The longer the sequences are, the more accurate the values of the correlation get to the actual value.

The acutal value of the correlation is not reached within a finite number of samples. For seq1=[1,-1,1,-1,...] the autocorrelation should switch between -1 and 1 since the sequence is either perfectly linear or perfectly antilinear. Still, the value deviation comes from the fact that I decided after some testing with the unittest to implement the Cov[X,Y] = E[XY]-E[X]*E[Y]. I implemented an additonal function that calculates the covariance with the 1/(n-1) factor and as you can see it is closer to the acutal value, also with a low k! The reason why I didnt use it in my implementation is that it failled the unittest.

For the second sequence the same observations as for the first sequence remain true only the values change.

4.4.1 Lag Size¶

correlation coefficient: measures the linear dependency between RVs. The covariance basically shows the same thing but without normalization, so it is more difficult to find a meaningful interpretation since the amplitude is harder to classify.

Lag Size: The autocorelation and lag size are negativly corrlated, meaning the larger the lag size, the smaller the autocorrelation. If rho is set to a highe value, the autocorelation decreses slower with a growing lag size. For the test sequences, seq1 a lag soize of 0,1 and for seq2 a lag size of 0,1,2 should be chosen. AS I said before the maximum lag should be lower than the period since the correlation coeffs then starts to repeat, thus no new information is revealed.

4.4.2 Cororelation Times in the DES¶

Yeez, that was a lot of work.

Correlation between IAT and waiting time of a packet: Are negative correlated. This makes sense since when the IAT gets big, less packets arrive, thus there are less packets in the buffer which leads to a smaller waiting time per packet. The correlation gets smaller for a bigger rho.

Correlation between IAT and service time of a packet: Basically zero, indicating no correlation accross all rho. This makes sense since the two exponential distributions where chosen to be independent

Correlation between IAT and system time (waiting time + service time) of a packet: Negatively correlated. It changes significantly with rho. For rho=0.01 it is basically 0. For rho=0.5 the correlation coeff. is -0.229. For rho=0.8 its -0.17 and for rho=0.95 its -0.08. this makes sense since the system time is a combination of the waiting and service time. For a small rho the service time plays the more significant role, thus the correlation is close to 0. When rho rises, the waiting time is more significant and we already established that there is a negative correlation between IAT and waiting time. The latter decreases with an further incearsing rho, as does the correlation between IAS and system time.

Correlation between service time and system time of a packet: Is positive, meaning when one increases so does the other. It overall decraeses with an increasing rho since the system time is the combination of waiting and service time, the former is heavily affected by a large rho but the serivice time is not, thus the correlation decreases with a large rho.

Auto-correlation of waiting time with lags ranging from 1 to 20 n: Is positive and decreases with a steadily increasing lag. For larger rho the negative slopes of the curves are lower. This makes sense because in this situation the server is just quick enough to handle the arriving packets. Thus if a packet has to wait it is higly likely that another packet has to wait to, since the server is barly quick enough to handle the packets the influence can be seen for a longer time, thus the correaltion remainson higher positive values for longer.
